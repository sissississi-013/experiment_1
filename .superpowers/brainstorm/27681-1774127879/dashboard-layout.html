A. Single-Page Flow

Everything on one scrolling page: intent input at top, live progress in the middle, results at bottom. Like a Keynote presentation — each section is a "slide". Clean, linear, Apple-style storytelling.

C. Tabbed Sections

iOS-style segmented control at top: New Run / History / Gallery. Each tab is a full-screen experience. Clean separation of concerns. The "New Run" tab has a form, History shows past runs, Gallery shows images across runs.
